Expense Tracker Application

A full-stack web app that helps users manage their finances by tracking income and expenses.

Backend:
-Built with Node.js, Express, and MongoDB.

-Provides RESTful APIs for CRUD operations on transactions (type, amount, description, category, date).

-Supports filtering by type, category, and date.

-Includes data validation and error handling for all endpoints.

Frontend:

-Developed with React.

-Features a dashboard displaying total income, expenses, and a transaction list.

-Users can add, view, and filter transactions easily.

-Displays visual charts (pie/bar) to compare income vs. expenses.

-Includes form validation, loading states, and error messages for better UX.
